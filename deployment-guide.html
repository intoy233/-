🚀 智能交互平台部署指南

恭喜！ 您的网站已经准备好部署了。选择下面任一方法即可让全世界访问您的网站。

方法一：Vercel 部署（推荐）

步骤 1：登录 Vercel

1. 访问 Vercel.com
2. 点击 "Sign Up" 或 "Login"
3. 使用 GitHub、GitLab 或 Email 登录

步骤 2：部署网站

方式 A：拖拽部署（最简单）
1. 登录后点击 "Add New..." → "Project"
2. 选择 "Browse All Templates" 下方的 "Deploy a new project"
3. 将整个 UI 文件夹拖拽到上传区域
4. 等待部署完成（约1-2分钟）
方式 B：命令行部署
在项目文件夹中打开命令行，运行：
vercel login
vercel --prod

方法二：Netlify 部署

1. 访问 Netlify.com
2. 注册/登录账户
3. 将整个 UI 文件夹拖拽到 "Deploy" 区域
4. 等待部署完成

方法三：GitHub Pages

1. 在 GitHub 创建新仓库
2. 上传所有文件到仓库
3. 在仓库设置中启用 GitHub Pages
4. 选择 main 分支作为源

⚠️ 部署后必须配置

重要：部署完成后，您需要更新 Supabase 配置：
  1. 登录 Supabase 控制台
  2. 进入您的项目 → Authentication → Settings
  3. 在 "Site URL" 中填入您的新域名
  4. 在 "Redirect URLs" 中添加：
    https://your-domain.com/auth.html
    https://your-domain.com/

🎉 部署完成后

测试清单：
  • ✅ 网站能正常加载
  • ✅ 用户注册功能正常
  • ✅ 用户登录功能正常
  • ✅ 密码重置功能正常
  • ✅ 在手机上显示正常

📱 获取访问链接

部署完成后，您将获得一个类似这样的链接：
https://your-project-name.vercel.app
https://your-project-name.netlify.app
https://username.github.io/repository-name
将此链接分享给任何人，他们都可以访问您的网站！
提示：如果遇到问题，请检查浏览器控制台的错误信息，通常是 Supabase 配置问题。
